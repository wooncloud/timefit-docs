# Configuration

Configure Timefit according to your needs.

## Basic Configuration

Create a configuration file in your project root:

```json
{
  "timefit": {
    "theme": "default",
    "autoSave": true,
    "notifications": true
  }
}
```

## Advanced Options

### Theme Customization

```json
{
  "theme": {
    "colors": {
      "primary": "#007acc",
      "secondary": "#f0f0f0"
    }
  }
}
```

### Performance Settings

```json
{
  "performance": {
    "cacheEnabled": true,
    "maxCacheSize": "100MB"
  }
}
```
