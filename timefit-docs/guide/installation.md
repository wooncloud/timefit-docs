# Installation

Learn how to install and set up Timefit.

## System Requirements

- Operating System: Windows, macOS, or Linux
- Node.js 16 or higher
- npm or yarn package manager

## Installation Steps

### Using npm

```bash
npm install timefit
```

### Using yarn

```bash
yarn add timefit
```

## Verification

To verify the installation:

```bash
timefit --version
```
